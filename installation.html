Installation

For now it it's only tested with Rails 2.3.8, and it would not work with other version We are working to support all rails version from 2.2.* to 3.*, and we make update when we ready

you need folder features

if you don't have folder features inside you rails application folder, it would be the best to boostrap cucumber inside you application, instruction are here

install cucumber fm

            script/plugin install git://github.com/cs3b/cucumber_fm.git

start application

            script/server

And open documentation features page, by default below link should work.
http://localhost:3000/documentation/features

it will ask you for password but for know it doesn't matter what you fill in just press OK

for further information how to use it got to Usage
