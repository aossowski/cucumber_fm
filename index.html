Cucumber Feature Manager

We are storing specifications for our projects as cucumber features, but we extend usage (look at example below) and add some new conventions to:

  • make management possible
  • improve collaboration with client
  • attach estimation direct to scenarios

so we were able to remove duplication of information that was splited across many places, we are continuously improving our approach and working hard to make it more user friendly in use.

It is still beta but if you wanna test then go ahead, your feedback would be valuable
